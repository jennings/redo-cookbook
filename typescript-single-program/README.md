# TypeScript Console Application Example

This example demonstrates how to build a simple TypeScript console application using redo.

## Files

- `index.ts` - The main TypeScript source file
- `tsconfig.json` - TypeScript configuration
- `package.json` - Node.js package configuration
- `all.do` - Redo build script

## Building

```bash
redo all
```

## Running

After building, run the application with:

```bash
node index.js
```
