# C# Console Application Example

This example demonstrates how to build a simple C# console application using redo.

## Files

- `Program.cs` - The main C# source file
- `HelloWorld.csproj` - The C# project file
- `all.do` - Redo build script

## Building

```bash
redo all
```

## Running

After building, run the application with:

```bash
dotnet run
```
